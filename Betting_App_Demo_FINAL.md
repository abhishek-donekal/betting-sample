## üéØ PROJECT OVERVIEW

A **professional sports betting application demo** with modern Hard Rock-inspired design. The application includes user registration, live sports betting, credit management, and an admin dashboard.

### Key Features

- User registration and login
- Live sports odds from multiple leagues (NBA, NFL, NCAA, WNBA, CFL)
- Credit-based betting system ($100 starting credits)
- Parlay betting capabilities
- Betting history tracking
- Admin dashboard for management
- Mobile-responsive design (works on phones, tablets, desktops)
